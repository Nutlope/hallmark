What Flarepin captures

Uncaught exceptions

Full stack traces with source context, request metadata, and environment state at the moment of failure.

Slow queries

Database call profiling with query text, parameters, and execution time. Postgres, MySQL, MongoDB, Redis.

Async-stack issues

Reconstructed async traces across promises, coroutines, and goroutines so you can follow execution across awaits.

Three runtimes

First-class SDKs for Node, Python, and Go. Minimal overhead, no monkey-patching surprises.

Self-hosted or managed

Run Flarepin on your own infrastructure with a single binary, or use our managed service.

18ms ingestion

Median ingestion latency measured across our managed cluster. Events are queryable in under a second.

Install in two lines

Drop the SDK in, set your project key, and uncaught exceptions are captured automatically.

npm install flarepin

import { Flarepin } from 'flarepin';
Flarepin.init({ key: process.env.FLAREPIN_KEY });

Python and Go SDKs follow the same shape. See the docs for framework integrations.
